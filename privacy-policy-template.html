🔐 Base Encoder 隱私權政策

最後更新：2024年12月

重點摘要： Base Encoder 是一個本地運行的 Chrome 擴充功能，所有資料處理都在您的設備上進行，不會傳送到任何遠端伺服器。

📊 資料收集

Base Encoder 僅在您的本地設備上儲存以下資料：

  • 使用者輸入內容：您輸入的文字內容（用於編碼和解碼功能）
  • 介面設定：您的偏好設定，如標籤頁狀態、介面配置等
  • 使用記錄：最近使用的編碼類型和設定

🎯 資料使用目的

我們收集和使用這些資料的唯一目的是：

  • 提供編碼和解碼功能
  • 改善使用者體驗（記住您的設定）
  • 讓您可以在重新開啟擴充功能時繼續之前的工作

💾 資料儲存

  • 儲存位置：所有資料僅儲存在您的本地設備上
  • 儲存方式：使用 Chrome 瀏覽器的本地儲存 API
  • 資料傳輸：資料不會傳送到任何遠端伺服器
  • 第三方存取：無任何第三方可存取您的資料

🔒 資料安全

我們採取以下措施保護您的資料：

  • 使用 Chrome 瀏覽器的安全儲存機制
  • 遵循 Chrome 擴充功能的安全最佳實務
  • 採用最小權限原則，僅請求必要的權限
  • 定期更新以修復安全漏洞

🚫 我們不會做的事

  • ❌ 不會將您的資料傳送到外部伺服器
  • ❌ 不會與第三方分享您的資料
  • ❌ 不會販售您的個人資訊
  • ❌ 不會追蹤您的瀏覽行為
  • ❌ 不會收集與擴充功能功能無關的資料

🛠️ 資料控制

您對自己的資料擁有完全控制權：

  • 查看資料：您可以隨時在擴充功能中查看儲存的內容
  • 刪除資料：您可以隨時清除儲存的資料
  • 匯出資料：您可以複製和保存您的內容
  • 完全移除：解除安裝擴充功能會自動刪除所有相關資料

📱 權限說明

Base Encoder 僅請求以下權限：

  • storage：用於在本地儲存您的輸入內容和設定
重要：我們不會請求任何網路存取權限，確保您的資料不會離開您的設備。

🔄 政策更新

我們可能會不定期更新此隱私權政策。重大變更會通過以下方式通知您：

  • 在擴充功能中顯示通知
  • 更新此頁面的「最後更新」日期

⚖️ 法律合規

此隱私權政策符合以下規範：

  • Chrome Web Store 開發者政策
  • Google Privacy Policy 要求
  • 一般資料保護規則 (GDPR) 原則
